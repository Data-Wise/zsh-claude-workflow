# Create Course Syllabus

Generate a comprehensive course syllabus with all required components.

## Purpose

Create professional syllabi that include:
- Course information and objectives
- Grading policies and assignments
- Weekly schedule and topics
- Required materials and resources
- University policies and accommodations

## Required Input

- **Course title and number**
- **Semester and year**
- **Meeting times and location**
- **Learning objectives** (or topic area to generate them)
- **Assessment structure** (exams, assignments, participation)

## Process

1. Gather course information
2. Define or refine learning objectives
3. Create weekly topic schedule
4. Specify assessment weights and policies
5. Include required institutional statements
6. Format for distribution (PDF-ready markdown)

## Output Format

```markdown
# [Course Number]: [Course Title]
## [Semester Year]

### Instructor Information
- **Instructor**: [Name]
- **Office**: [Location]
- **Office Hours**: [Times]
- **Email**: [Email]

### Course Description
[Description text]

### Learning Objectives
By the end of this course, students will be able to:
1. [Objective 1]
2. [Objective 2]
...

### Required Materials
- [Textbook/Resources]

### Grading
| Component | Weight |
|-----------|--------|
| Exams | X% |
| Assignments | X% |
| Participation | X% |

### Weekly Schedule
| Week | Topic | Readings | Due |
|------|-------|----------|-----|
| 1 | [Topic] | Ch. X | |
...

### Course Policies
[Attendance, late work, academic integrity]

### University Policies
[ADA accommodations, Title IX, etc.]
```

## Customization Options

- **Format**: Standard, compressed, detailed
- **Institution**: Include specific policy templates
- **Accessibility**: Screen-reader friendly formatting

## Tips

- Align objectives with Bloom's taxonomy verbs
- Be specific about grade cutoffs
- Include clear communication expectations
- Provide Canvas/LMS integration notes

## Examples

**Request**: "Create a syllabus for STAT 101 Introduction to Statistics, Fall 2025, MWF 10-10:50am"

**Request**: "Generate a graduate seminar syllabus on Bayesian Methods, 15 weeks"

**Request**: "Update my existing syllabus to add a new project component worth 20%"
