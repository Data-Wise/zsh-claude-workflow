# Generate Student Feedback

Create constructive, personalized feedback for student submissions.

## Purpose

Generate helpful feedback that:
- Identifies strengths and areas for improvement
- Provides specific, actionable suggestions
- Maintains encouraging and professional tone
- Supports student learning and growth

## Required Input

- **Assignment type**: Exam, paper, code, presentation
- **Student work**: The submission to evaluate
- **Rubric** (optional): Grading criteria to reference
- **Grade** (optional): If already determined

## Process

1. Review student submission thoroughly
2. Identify key strengths to acknowledge
3. Note areas needing improvement
4. Provide specific, constructive suggestions
5. Offer resources or next steps when appropriate
6. Maintain supportive, growth-oriented tone

## Output Format

```markdown
## Feedback for [Assignment Name]

### Strengths
- [Specific positive observation 1]
- [Specific positive observation 2]

### Areas for Improvement
- [Issue 1]: [Specific suggestion for improvement]
- [Issue 2]: [Specific suggestion for improvement]

### Detailed Comments
[Section-by-section or problem-by-problem feedback]

### Summary
[Overall assessment and encouragement]

### Suggested Resources
- [Resource 1 if applicable]
- [Resource 2 if applicable]

**Grade**: [X/Y] or [Letter Grade]
```

## Feedback Types

### For Written Work
- Thesis clarity and argument structure
- Evidence and support quality
- Writing mechanics and style
- Citation and formatting

### For Code/Technical Work
- Correctness and functionality
- Code style and readability
- Efficiency considerations
- Documentation quality

### For Exams
- Conceptual understanding
- Problem-solving approach
- Common misconceptions addressed
- Study recommendations

## Tone Guidelines

- Start with genuine positives
- Be specific, not vague ("good job")
- Frame criticism constructively
- Use "consider" rather than "you should"
- End with encouragement

## Examples

**Request**: "Provide feedback on this student's regression analysis assignment [paste work]"

**Request**: "Generate feedback for exam where student scored 72/100, struggled with hypothesis testing"

**Request**: "Create encouraging feedback for a first draft of a methods section"

## Batch Feedback

For multiple submissions:
- Maintain consistency across students
- Note common issues for class-wide addressing
- Individualize specific comments
